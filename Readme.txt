Framework developed with Javascript ES6 + WebdriverIO + Allure reports


1) After downloading GIT repository, execute npm install so all the node module dependencies are downloaded.

    > npm install

2) In the terminal to execute the feature file, below command is used. 

    > npx wdio .\wdio.conf.js

3) To get allure reports, execute below command  
    
    > allure generate allure-results
    > allure open
